# 攀爬架改造

新年伊始，觉得应该做点改变，娃娃的攀爬架搭建起来有半年了，但是架构一直没有发生过变化，对于他来说，现在难度已经变低了，于是考虑拆了，重新搭建一个攀爬难度系数更高的架构

我和老婆负责拆卸，搭建新的架构由我来负责，娃娃的外婆将零件重新擦拭了一遍。

说干就干，上午完成了拆卸和擦拭的工作。下午安排组装。

幸好厂商提供新的组装图纸，不然凭借我现在的空间想象能力，完成重新组织的难度超级高。

惯用的解决方案就是动态规划，将整体拆分为多个子模块，整体由滑梯，三级阶梯，阶梯承载模块，屋顶，平台挡板组成。而我要做的就是先将这些子的模块拼接好，然后完成最终整体的拼接。小零件分为管道，连通器和螺丝组成。


## 什么是寄存器 (What)?

寄存器本质就是一块内存，是 CPU 可以读写的最快的存储器件，是 CPU 的一部分，用于存储 CPU 运行时的数据。

## 为什么要使用寄存器 (Why)?

减少内存访问次数，提高程序运行效率。

## 什么时候使用寄存器 (When)?

- 临时存储数据
- 传递参数
- 保存返回地址
- 保存状态
- 保存临时结果
- 保存局部变量
- 保存全局变量
- 保存中间结果
- 保存常量
- 保存标志位
- 保存控制信息

## 什么地方使用寄存器 (Where)?

- CPU
- GPU

## 谁使用寄存器 (Who)?

- CPU
- GPU

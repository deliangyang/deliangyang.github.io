## Parser
- 语法分析
- 根据给定的形式文法对有单词序列构成的输入文本进行分析，并确定语法结构的一种过程。
- 单词流作为输入。
- 语法检查、并构建由输入的单词组成的数据结构。
  - 语法分析树
  - 抽象语法树
- 如何从语法的起止符号推导出输入符号串
  - 自顶向下分析
    - 搜索输入符号串的最左推导。单词从左到右依次使用
  - 自底向上分析
    - 根据给定的形式语法规则进行改写，最后改写为语法的起始符号
